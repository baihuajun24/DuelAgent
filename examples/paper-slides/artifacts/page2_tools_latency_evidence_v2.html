Message for the Audience / 想让听众立刻记住的话

  • Agents are not just “LLMs that generate slowly”.真正拖慢 end-to-end latency 的，往往是反复的 tool loop。
  • Across papers, tools repeatedly appear as a first-order latency term.不只是本文自己这么说，其他工作也反复测到工具执行是主要组成部分。
  • So the right optimization target is overlap, not only faster decoding.因此要优化的是 “LLM thinking + tool execution” 的重叠，而不只是 token decode。
